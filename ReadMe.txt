== Description ==

Introducing the VedicAstroAPI Wordpress plugin - a free and easy-to-use tool to integrate astrological predictions and kundli analysis into your Wordpress website. This plugin allows you to easily integrate <a href="https://documenter.getpostman.com/view/16788527/VUjMq6ts/">VedicAstroAPI's REST APIs</a> via shortcodes, giving you the flexibility to customize your astrological content using inbuilt form fields or your own forms.

Please note that while the plugin itself is free, it uses the APIs from <a href="https://vedicastroapi.com/">VedicAstroAPI.com</a>, which are chargeable under the <a href="https://vedicastroapi.com/pricing/"> Basic and Pro plans</a>. The Basic Plan offers 50,000 API calls per month, and the Pro Plan provides all APIs with multi-lingual support in 7 additional languages and has the same limit.

Please refer to <a href="https://vedicastroapi.com/">vedicastroapi.com</a> for more details on the <a href="https://documenter.getpostman.com/view/16788527/VUjMq6ts/">API documentations</a>, including <a href="https://vedicastroapi.com/pricing/">pricing and usage limits</a>. Please note that the older version of the plugin has been deprecated.

Features of VedicAstroAPI include:

7+ Languages (Indian & International)
*   Predictions - Daily & Weekly predictions, Yearly, BioRythm, Numerology, Day Number
*   Horoscope & Kundli - Planet Details, Chart Image (SVG), Personal Characteristics, Divisional Charts, Ashtakvarga
*   Matching - North Indian (Ashtakoot), South Indian (Dashakoot), AI Aggregate Match, Papasamaya Match, Western Match, Bulk Profile Matching
*   Extended Horoscope - Extended Kundli Details,  Current Sade Sati, Gem Suggestion, Numero Table, Rudraksh Suggestion, Varshapal Details
*   Dashas - Mahadasha, Antardasha, Paryantar Dasha, Current Mahadasha, Specific Sub-dasha Finder, Char Dasha, Yogini Dasha
*   Panchang - Panchang, Chogadiya & Hora Muhurta, Moon Phases, Sun rise, Noon & Set, Moon Rise & Moon Set, Retrograde Dates
*   Doshas - Mangal Dosh, Kaal-Sarp Dosh, Manglik Dosh, Pitra Dosh, Papasamaya Dosh
*   Utilities - Gem-stone details, City lat-lon Search

Please refer to <a href="https://vedicastroapi.com/">vedicastroapi.com</a> for more information and documentation on these features.

Note - If you are using the old plugin and are replacing with the new one

== Installation ==

1. Upload the 'vedic-astro-api' folder to the directory '/wp-content/plugins/'.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. You can adjust the necessary settings using your WordPress admin panel in 'VedicAstroAPI' section.
